SEP10 Freedom Project

Contextual Information

Overview of the project and the objective

For my SEP10 Freedom Project, I explored Graphic Design and how digital tools can be used to create visual content. The goal of the project was to learn a new skill, research the field, and build a website that explains what I learned. Throughout the project, I practiced problem-solving, creativity, and coding while documenting my progress.

Content

What's on my website

My website includes information about Graphic Design, the tools and resources I used, my learning process, and the projects I created. It also shows the code I worked on, screenshots of my progress, and reflections about what I learned throughout the year.

Challenges

What went wrong and how I handled it

One of the biggest challenges was understanding new coding concepts and fixing problems when my website did not look the way I expected. Sometimes small mistakes in my code caused bigger issues, but by testing different solutions, asking questions, and staying patient, I was able to improve my project.

Takeaways

  • Planning my work made coding easier.
  • I became more confident using HTML and CSS.
  • Problem-solving is an important part of programming.
