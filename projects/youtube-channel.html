SEP10 Freedom Project

Contextual Information

Overview of the project and the objective

After learning HTML and CSS for two months, Mr. Mueller wanted to test our skills by giving us a project. The prompt required creating a webpage that promotes a fake movie using headings, paragraphs, spans, divs, and images. Understanding spacing, alignment, and styling was crucial. I worked with my friend Jasmina, who was flexible and easy to work with.

Content

What's on my website

The webpage required Creative Commons images, headings, paragraphs, divs, spans, input fields, submit buttons, selectors, and box model properties. Planning the site in markdown helped break tasks into manageable parts. My partner and I split the work evenly.

Challenges

What went wrong and how I handled it

Centering images and aligning divs were the most frustrating parts. Adjusting sizes and layout often caused unexpected changes, but through trial and error we learned how CSS positioning works.

Takeaways

  • Planning saves time and improves clarity.
  • Collaboration made the work easier.
  • Clear communication improved the final result.

Next Steps

  • Improve layout and spacing.
  • Refine CSS for cleaner structure.
  • Add more detail to the fictional movie.
  • Experiment with gradients and visuals.
  • Reduce clutter for better readability.
