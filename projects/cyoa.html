Contextual Information

Overview of the project and the objective

After learning about Git & Github for the past 2-3 months, Mr. Mueller decided to give us a project that involves collaboration through Github towards creating an adventure that you can choose the pathway. This project was created with my friend Stacy, working with her was very fun & I loved working with her.

Basically, I created the Male Celebrity perspective where the male celebrity goes on this crazy adventure, and theres only one winning way for him.
